# Test Data Directory

This directory contains test data files for integration testing.

## Image Test Files

- `test_notes.jpg`: Sample handwritten or printed notes for OCR testing
- `blurry_image.jpg`: Low-quality image for edge case testing
- `dark_image.jpg`: Low-light image for edge case testing

## Usage

Test images can be created programmatically in tests, or you can add sample images here for consistent testing.
